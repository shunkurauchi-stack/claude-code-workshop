# Claude Code 入門 — 社内勉強会スライド

## プロジェクト概要

Claude Code の入門勉強会（社内向け）で使うスライド資料。
単一の HTML ファイル (`claude-code-intro.html`) にすべての CSS・HTML・JavaScript が含まれる、スタンドアローンなプレゼンテーション。

## ファイル構成

```
claude-code-intro/
└── claude-code-intro.html   # スライド本体（全11スライド）
```

## スライド構成

| スライド | 内容 |
|---------|------|
| 1 | タイトル |
| 2 | アジェンダ（本日の流れ） |
| 3 | Claude Code とは？ |
| 4 | 普通のAIチャットとの違い（比較表） |
| 5 | インストールと初期設定 |
| 6 | 基本的な使い方 ① 指示の出し方 |
| 7 | 基本的な使い方 ② チームで活かす機能（CLAUDE.md 等） |
| 8 | 業務での活用事例（新機能実装・バグ修正・ドキュメント・テスト） |
| 9 | デモタイム |
| 10 | うまく使うコツ（Tips） |
| 11 | まとめ／次のステップ |

## 技術仕様

- 純粋な HTML/CSS/JavaScript（外部依存なし）
- ブラウザで直接開いて使用可能
- キーボード操作対応（← → ↑ ↓ キーでスライド切り替え）
- ナビゲーションドットでスライドへのジャンプも可能
- 全11スライド構成

## デザイン

- ダークテーマ（ネイビー系背景）
- オレンジ (`#D97757`) をアクセントカラーとして使用
- CSS 変数で色管理（`:root` に定義）
- 日本語フォント優先（Hiragino Sans / Noto Sans JP）

## 開発時の注意

- すべて1ファイルに収まっているため、スライドの追加・削除時は JS の `totalSlides` 変数 (`const totalSlides = 11`) も更新すること
- 新しいスライドは `id="slide-N"` の形式で追加し、既存のスライドクラス（`slide-title`, `slide-content`, `slide-summary` 等）を流用する
